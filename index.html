SECTIONS 08 & 09

Admin & Quick Reference

Configure settings, manage staff, protect your data, and use daily checklists to stay on track.

Admin & Backups

Membership Plans: Set default prices in Admin > Membership Plans (e.g., Basic at 1,500/mo).

Auto-Block Rules: Create rules to block members automatically after X unpaid months, with grace periods.

Staff Accounts: Assign roles ("Staff" for limited access, "Admin" for full access).

Backups: Your most important task. Locally store database via Admin > Backup & Reset. Save your JSON file manually to a USB or Cloud Drive.

Daily Checklist

Morning: Open FitManager, check Dashboard. Note pending/blocked members.

During Day: Add new members, record received payments immediately.

Evening: Check Today's Collection on Dashboard. Send WhatsApp Reminders. Perform Admin Backup and close the app.

Frequently Asked Questions

Does it need internet?

No. FitManager works 100% offline after installation. Your data never leaves your computer.

Can I use on multiple PCs?

Each computer needs its own license. The key is tied to your specific hardware.

What does "Pending" mean?

Member has unpaid months. Check the "Due Amount" column to see how much they owe.

What is "Frozen" status?

Membership is paused. No billing or reminders during freeze (travel, injury, etc.).

Can members pay in advance?

Yes! Enter multiple months in the "Months Paid" field when recording payment.

How to move data to new PC?

Backup on old PC → Install on new PC → Get new license → Import backup file.
